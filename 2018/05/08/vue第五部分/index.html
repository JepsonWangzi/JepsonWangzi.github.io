##路由：
根据不同的hash值，为页面对应显示不同的组件。

  • 路由规则：hash值是如何对应组件的

  • vue-rooter的使用

    1：指定路由规则
2：指定显示的位置

  • 步骤：
    1：创建vm实例
    2：创建路由模板
    3：配置路由的规则，创建路由对象
    4: 要将路由对象和vm实例关联起来
    5: 给路由匹配到的组件指定显示的位置 指的就是组件的占位符

##路由中参数的获取；

  • 1：用？传递的参数在vue-router的获取
    例子： http://xx.com/index.html#/detail?id=123
    参数信息 id=123;
    获取id；

  • 在创建路由对象的时候可以通过this.$route获取所有信息

    //path  路径信息
//hash  hash值信息(模拟出来的)
//query ?传递的参数信息
//name  路由的名字
//params 路由参数！
  • 用this.$route.query.id 就可以获取id了；
